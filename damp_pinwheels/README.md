# damp_pinwheels

Разреженное битовое кодирование и раскладка DAMP для карты пинвилов 360 углов × 5 позиций.

## Запуск

```
python3 -m damp_pinwheels --angles 360 --positions 5 --output pinwheel_output --log-every 100 --snapshot-every 50
```

Выходные файлы:
- `pinwheel_layout.ppm` (карта ориентаций, оттенок = угол)
- `pinwheel_layout.csv` (координаты в сетке + метаданные)

## Примечания

- Кодирование угла использует компонентное назначение из DAMP: sin(угол) + cos(угол) + радиус.
- Значения радиуса равномерно распределены в [0, 1].
- Раскладка выполняется дальним проходом, затем ближним, по DAMP.
- `--log-every` управляет частотой вывода прогресса в консоль (например, 1 для каждого шага).
- `--snapshot-every` определяет, как часто обновляется `pinwheel_layout.ppm` во время раскладки.
